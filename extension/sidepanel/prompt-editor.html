Customize how the LLM generates IXML grammars for form parsing

Multi-Grammar Architecture

This prompt instructs the LLM to generate multiple specialized grammars instead of one complex grammar. Each grammar extracts one type of element:

  • input-only - Extracts INPUT elements
  • select-only - Extracts SELECT/dropdown elements
  • textarea-only - Extracts TEXTAREA elements

The results are combined in a multi-pass parsing pipeline.
