🔬 Linearized Table Parsing POC

Demonstrating IXML parsing of visible table text (what users see) instead of HTML markup.

Concept: Extract the linearized text representation of an HTML table, then parse it with an IXML grammar to produce structured XML. This approach leverages IXML's strength in parsing unstructured text patterns.

Example: Contact List Table

Name Email Phone
John Doe john@example.com 555-1234
Jane Smith jane@example.com 555-5678
Bob Johnson bob@example.com 555-9999
